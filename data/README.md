# Chest X-Ray Images (Pneumonia) Dataset

Download the dataset from [Kaggle](https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia/data), and place it in this `data/` directory. The structure should look like this:

```bash
data/
└── chest_xray
    ├── train
    │   ├── NORMAL
    │   └── PNEUMONIA
    ├── val
    │   ├── NORMAL
    │   └── PNEUMONIA
    └── test
        ├── NORMAL
        └── PNEUMONIA
```
